# 程式語言教學平台 🚀

一個現代化、美觀且功能完整的程式語言學習平台，專為初學者設計，提供 HTML、Python、JavaScript 和 Java 的完整教學內容。

## 📊 教學內容統計
- 🌐 **HTML**：6 課完整基礎教學
- 🐍 **Python**：14 課從入門到進階
- 🟨 **JavaScript**：12 課前端開發完整教學
- ☕ **Java**：6 課程式設計基礎
- 📝 **總計**：38 課豐富教學內容
- 🔄 **程式碼範例**：100+ 個實用範例，全部支援一鍵複製

## ✨ 功能特色

### 🎯 核心功能
- **多語言支援**：提供 HTML、Python、JavaScript、Java 四種核心技術教學
- **零基礎友善**：從最基礎的概念開始，循序漸進的學習路徑
- **互動式學習**：豐富的程式範例和實作練習
- **響應式設計**：支援桌面、平板、手機等各種裝置

### 🎨 使用者體驗
- **深淺色主題**：支援明亮/暗黑模式切換，保護眼睛
- **現代化介面**：使用 Tailwind CSS 打造美觀直覺的設計
- **流暢動畫**：平滑的過場效果和互動回饋
- **可展開課程**：點擊展開詳細內容，節省頁面空間
- **程式碼複製**：一鍵複製功能，方便實作練習

### 📚 教學內容
- **HTML**：網頁結構基礎，從零開始學習標籤和語義化
- **Python**：適合初學者，涵蓋完整語法和程式設計概念
- **JavaScript**：網頁開發核心，學習前端互動和現代語法
- **Java**：企業級開發，物件導向程式設計基礎

## 🛠️ 技術架構

### 前端技術
- **HTML5**：語義化標籤，提升可訪問性

## 📦 安裝與使用

### 直接使用 HTML 版本
```bash
# 克隆專案
git clone [your-repo-url]
cd programming-tutorial

# 直接開啟 index.html
# 使用瀏覽器開啟 index.html 檔案即可
```

### 🎯 使用指南
1. **選擇語言**：從首頁選擇想學習的程式語言 (HTML/Python/JavaScript/Java)
2. **逐步學習**：按照課程順序，從基礎到進階逐步學習
3. **複製程式碼**：點擊程式碼區塊右上角的複製按鈕，快速複製範例程式碼
4. **實作練習**：將複製的程式碼貼到您的編輯器中進行實作
5. **主題切換**：使用右上角的主題切換按鈕，選擇適合的顯示模式

### React 版本開發環境
```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

# 預覽建置結果
npm run preview
```
