**Code ED** 是一個專為華語學習者設計的現代化程式設計教學平台。我們提供從零基礎到進階的完整學習路徑，涵蓋 7 種主流程式語言，共 39 堂精心設計的課程。

### 🚀 核心優勢

- **� 零基礎友善**：專為初學者設計，無需任何程式設計背景
- **📈 漸進式學習**：從基礎語法到進階應用，循序漸進的學習路徑
- **� 實戰導向**：每個概念都配有豐富的程式碼範例和實作練習
- **🎨 現代化介面**：美觀的 UI 設計，支援深色模式和一鍵複製
- **⚡ 即開即用**：無需安裝任何軟體，瀏覽器即可開始學習
- **� 持續更新**：定期新增內容，跟上技術發展趨勢

## 🛠️ 技術架構

- **HTML5**：語義化標籤，提升可訪問性
- **CSS3**：現代化樣式，支援明暗主題
- **JavaScript (ES6+)**：原生 JavaScript，無框架依賴
- **Tailwind CSS**：實用優先的 CSS 框架

## ✨ 核心功能

- **多語言支援**：7 種程式語言完整教學
- **分級學習**：初學者→中級→進階的學習路徑
- **現代化圖標**：全 SVG 設計，漸層效果，官方色彩
- **響應式設計**：完美適配各種裝置
- **智能導航**：按難度分類的下拉式選單

## 📦 快速開始

### 🌐 線上體驗
直接訪問：**[立即體驗](https://xingch0405.github.io/Code-ED/)**

### 💻 本地安裝
```bash
# 克隆專案
git clone https://github.com/xingch0405/Code-ED.git
cd Code-ED

# 使用瀏覽器開啟 index.html
# 或使用 Live Server 擴充功能 (推薦)
```

### 🎯 使用方法
1. **選擇語言** → 從首頁點擊想學習的程式語言
2. **導航切換** → 使用下拉選單按難度查看所有語言
3. **逐步學習** → 點擊課程標題展開詳細內容
4. **複製程式碼** → 點擊程式碼區塊的複製按鈕
5. **切換主題** → 使用右上角的主題切換按鈕

## 🎯 學習路徑

- **💜 C# (6 課)**：微軟 .NET 生態系統，企業級開發
- **☕ Java (8 課)**：跨平台執行，物件導向程式設計
- **🔧 C 語言 (5 課)**：系統程式設計，記憶體管理

### 🔴 進階路徑

- **⚡ C++ (6 課)**：高效能程式設計，STL 標準庫
