Python 教學

最適合初學者的程式語言

學習目標

  • • 了解 Python 基本語法和概念
  • • 掌握變數、資料型別和運算
  • • 學會條件判斷和迴圈控制
  • • 能夠編寫簡單的 Python 程式

1. Python 簡介與安裝

基礎

了解 Python 是什麼，為什麼選擇 Python，以及如何安裝 Python 開發環境。

程式碼範例：

# 這是你的第一個 Python 程式
print("Hello, World!")
print("歡迎來到 Python 的世界！")

詳細說明：

Python 是一種高階程式語言，以其簡潔易讀的語法而聞名。它被廣泛應用於：

  • 網頁開發
  • 數據科學與分析
  • 人工智慧與機器學習
  • 自動化腳本
  • 科學計算

print() 函數用於在螢幕上顯示文字。這是最基本也是最常用的函數之一。

2. 變數與資料型別

基礎

學習如何宣告變數，以及 Python 中的基本資料型別。

程式碼範例：

# 變數宣告
name = "小明"           # 字串 (string)
age = 25              # 整數 (integer)
height = 175.5        # 浮點數 (float)
is_student = True     # 布林值 (boolean)

# 顯示變數內容
print(f"姓名：{name}")
print(f"年齡：{age}")
print(f"身高：{height} 公分")
print(f"是學生：{is_student}")

詳細說明：

Python 中的基本資料型別包括：

  • 字串 (str)：用引號包圍的文字
  • 整數 (int)：沒有小數點的數字
  • 浮點數 (float)：有小數點的數字
  • 布林值 (bool)：True 或 False

f-string 是格式化字串的現代方法，使用 f"" 並在大括號中放入變數。

JavaScript 教學

網頁開發的核心語言

學習目標

  • • 掌握 JavaScript 基本語法
  • • 了解變數、函數和資料結構
  • • 學會操作陣列和物件
  • • 能夠編寫互動式網頁功能

1. JavaScript 簡介

基礎

了解 JavaScript 的用途、特色，以及如何在網頁中使用。

程式碼範例：

// 這是你的第一個 JavaScript 程式
console.log("Hello, World!");
console.log("歡迎來到 JavaScript 的世界！");

// 在網頁中顯示訊息
alert("歡迎學習 JavaScript！");

// 修改網頁內容
document.getElementById("demo").innerHTML = "Hello JavaScript!";

詳細說明：

JavaScript 是網頁開發的核心語言，主要用於：

  • 網頁互動效果
  • 動態內容更新
  • 表單驗證
  • 後端開發 (Node.js)
  • 行動應用開發

console.log() 用於在開發者工具中顯示訊息，alert() 會彈出對話框。

Java 教學

企業級開發的首選語言

學習目標

  • • 了解 Java 物件導向概念
  • • 掌握基本語法和資料結構

1. Java 簡介與環境設置

基礎

了解 Java 的特色、應用領域，以及如何設置開發環境。

程式碼範例：

// 你的第一個 Java 程式
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
        System.out.println("歡迎來到 Java 的世界！");
    }
}

詳細說明：

Java 是一種物件導向的程式語言，具有以下特色：

  • 跨平台執行（Write Once, Run Anywhere）
  • 強型別系統
  • 自動記憶體管理
  • 豐富的標準函式庫
  • 企業級應用開發

每個 Java 程式都需要一個 main 方法作為程式的入口點。
