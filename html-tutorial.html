第1課：HTML 簡介與基本結構

基礎

了解什麼是 HTML，以及 HTML 文件的基本結構和必要元素。

什麼是 HTML？

HTML（HyperText Markup Language）是超文本標記語言，是建立網頁的標準語言。它使用標籤來描述網頁的結構和內容。

  • 超文本：可以包含連結到其他文件的文字
  • 標記語言：使用標籤來標記內容的結構
  • 網頁基礎：所有網頁都是用 HTML 建立的

基本 HTML 文件結構：

<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的第一個網頁</title>
</head>
<body>
    <h1>歡迎來到我的網頁！</h1>
    <p>這是我的第一個 HTML 頁面。</p>
</body>
</html>

結構說明：

  • <!DOCTYPE html>：告訴瀏覽器這是 HTML5 文件
  • <html>：根元素，包含整個網頁內容
  • <head>：包含網頁的元資訊（不會顯示在頁面上）
  • <meta charset="UTF-8">：設定字元編碼為 UTF-8
  • <title>：設定網頁標題（顯示在瀏覽器標籤上）
  • <body>：包含網頁的可見內容

第2課：文字標籤與格式化

基礎

學習如何使用各種文字標籤來建立標題、段落和格式化文字。

標題標籤（h1-h6）：

<h1>最大的標題</h1>
<h2>第二級標題</h2>
<h3>第三級標題</h3>
<h4>第四級標題</h4>
<h5>第五級標題</h5>
<h6>最小的標題</h6>

段落和文字格式：

<!-- 段落 -->
<p>這是一個段落。段落會自動換行，並且段落之間會有間距。</p>
<p>這是另一個段落。</p>

<!-- 文字格式化 -->
<p>這段文字包含 <strong>重要內容</strong> 和 <em>強調內容</em>。</p>
<p>這裡有 <b>粗體文字</b> 和 <i>斜體文字</i>。</p>
<p>這是 <u>底線文字</u> 和 <s>刪除線文字</s>。</p>

<!-- 換行和水平線 -->
<p>第一行<br>第二行</p>
<hr>
<p>水平線上方的內容</p>

重要概念：

  • <strong> vs <b>：<strong> 表示重要性，<b> 只是視覺上的粗體
  • <em> vs <i>：<em> 表示強調，<i> 只是視覺上的斜體
  • <br>：強制換行（自閉合標籤）
  • <hr>：水平分隔線（自閉合標籤）

第3課：連結與圖片

中級

學習如何建立連結和插入圖片，讓網頁更加豐富和互動。

連結標籤（<a>）：

<!-- 外部連結 -->
<a href="https://www.google.com">前往 Google</a>
<a href="https://www.google.com" target="_blank">在新視窗開啟 Google</a>

<!-- 內部連結 -->
<a href="about.html">關於我們</a>
<a href="#section1">跳到第一節</a>

<!-- 特殊連結 -->
<a href="mailto:someone@example.com">發送郵件</a>
<a href="tel:+886912345678">撥打電話</a>
<a href="sms:+886912345678">發送簡訊</a>

<!-- 下載連結 -->
<a href="document.pdf" download>下載 PDF 文件</a>

圖片標籤（<img>）：

<!-- 基本圖片 -->
<img src="photo.jpg" alt="美麗的風景照片">

<!-- 設定尺寸的圖片 -->
<img src="logo.png" alt="公司標誌" width="200" height="100">

<!-- 響應式圖片 -->
<img src="banner.jpg" alt="橫幅圖片" style="width: 100%; height: auto;">

<!-- 圖片連結 -->
<a href="https://www.example.com">
    <img src="clickable-image.jpg" alt="點擊這張圖片">
</a>

<!-- 圖片與文字說明 -->
<figure>
    <img src="artwork.jpg" alt="藝術作品">
    <figcaption>這是一幅美麗的藝術作品</figcaption>
</figure>

重要屬性說明：

  • href：連結的目標位址
  • target="_blank"：在新視窗或新標籤頁開啟連結
  • src：圖片的來源路徑
  • alt：圖片的替代文字（重要！用於無障礙設計）
  • width/height：設定圖片的寬度和高度
  • download：提示瀏覽器下載檔案而不是開啟

第4課：清單與導航

中級

學習建立各種類型的清單，以及如何建立網站導航選單。

無序清單（<ul>）：

<h3>我的興趣</h3>
<ul>
    <li>閱讀</li>
    <li>旅行</li>
    <li>攝影</li>
    <li>程式設計</li>
</ul>

有序清單（<ol>）：

<h3>學習步驟</h3>
<ol>
    <li>學習 HTML 基礎</li>
    <li>學習 CSS 樣式</li>
    <li>學習 JavaScript</li>
    <li>建立第一個網站</li>
</ol>

<!-- 不同編號類型 -->
<ol type="A">
    <li>選項 A</li>
    <li>選項 B</li>
</ol>

<ol type="I">
    <li>第一項</li>
    <li>第二項</li>
</ol>

巢狀清單與導航選單：

<!-- 巢狀清單 -->
<ul>
    <li>水果
        <ul>
            <li>蘋果</li>
            <li>香蕉</li>
            <li>橘子</li>
        </ul>
    </li>
    <li>蔬菜
        <ul>
            <li>胡蘿蔔</li>
            <li>青菜</li>
        </ul>
    </li>
</ul>

<!-- 導航選單 -->
<nav>
    <ul>
        <li><a href="index.html">首頁</a></li>
        <li><a href="about.html">關於我們</a></li>
        <li><a href="services.html">服務項目</a></li>
        <li><a href="contact.html">聯絡我們</a></li>
    </ul>
</nav>

第5課：表格 (Table)

進階

學習如何建立表格來展示結構化的資料。

基本表格結構：

<table>
    <thead>
        <tr>
            <th>姓名</th>
            <th>年齡</th>
            <th>職業</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>張三</td>
            <td>25</td>
            <td>工程師</td>
        </tr>
        <tr>
            <td>李四</td>
            <td>30</td>
            <td>設計師</td>
        </tr>
        <tr>
            <td>王五</td>
            <td>28</td>
            <td>產品經理</td>
        </tr>
    </tbody>
</table>

進階表格功能：

<table border="1">
    <caption>員工資料表</caption>
    <thead>
        <tr>
            <th rowspan="2">姓名</th>
            <th colspan="2">聯絡資訊</th>
            <th rowspan="2">部門</th>
        </tr>
        <tr>
            <th>電話</th>
            <th>Email</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>張三</td>
            <td>0912-345-678</td>
            <td>zhang@example.com</td>
            <td>技術部</td>
        </tr>
        <tr>
            <td>李四</td>
            <td>0987-654-321</td>
            <td>li@example.com</td>
            <td>設計部</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="4">總計：2 名員工</td>
        </tr>
    </tfoot>
</table>

表格標籤說明：

  • <table>：表格容器
  • <thead>：表格標題區域
  • <tbody>：表格主體內容
  • <tfoot>：表格頁尾區域
  • <tr>：表格列（table row）
  • <th>：表格標題儲存格
  • <td>：表格資料儲存格
  • colspan：跨欄合併
  • rowspan：跨列合併

第6課：表單 (Form)

進階

學習如何建立互動式表單，收集使用者輸入的資料。

基本表單元素：

<form action="/submit" method="post">
    <!-- 文字輸入框 -->
    <label for="name">姓名：</label>
    <input type="text" id="name" name="name" required>

    <!-- 電子郵件輸入框 -->
    <label for="email">電子郵件：</label>
    <input type="email" id="email" name="email" required>

    <!-- 密碼輸入框 -->
    <label for="password">密碼：</label>
    <input type="password" id="password" name="password" required>

    <!-- 數字輸入框 -->
    <label for="age">年齡：</label>
    <input type="number" id="age" name="age" min="1" max="120">

    <!-- 日期輸入框 -->
    <label for="birthday">生日：</label>
    <input type="date" id="birthday" name="birthday">

    <!-- 提交按鈕 -->
    <input type="submit" value="提交">
</form>

進階表單元素：

<form>
    <!-- 單選按鈕 -->
    <fieldset>
        <legend>性別：</legend>
        <input type="radio" id="male" name="gender" value="male">
        <label for="male">男性</label>

        <input type="radio" id="female" name="gender" value="female">
        <label for="female">女性</label>
    </fieldset>

    <!-- 複選框 -->
    <fieldset>
        <legend>興趣：</legend>
        <input type="checkbox" id="reading" name="hobbies" value="reading">
        <label for="reading">閱讀</label>

        <input type="checkbox" id="music" name="hobbies" value="music">
        <label for="music">音樂</label>

        <input type="checkbox" id="sports" name="hobbies" value="sports">
        <label for="sports">運動</label>
    </fieldset>

    <!-- 下拉選單 -->
    <label for="city">居住城市：</label>
    <select id="city" name="city">
        <option value="">請選擇</option>
        <option value="taipei">台北</option>
        <option value="taichung">台中</option>
        <option value="kaohsiung">高雄</option>
    </select>

    <!-- 多行文字區域 -->
    <label for="message">留言：</label>
    <textarea id="message" name="message" rows="4" cols="50" placeholder="請輸入您的留言..."></textarea>

    <!-- 檔案上傳 -->
    <label for="file">上傳檔案：</label>
    <input type="file" id="file" name="file" accept=".jpg,.png,.pdf">

    <!-- 按鈕 -->
    <button type="submit">提交表單</button>
    <button type="reset">重設</button>
    <button type="button">一般按鈕</button>
</form>

表單屬性說明：

  • action：表單提交的目標 URL
  • method：提交方式（get 或 post）
  • required：必填欄位
  • placeholder：輸入提示文字
  • min/max：數值範圍限制
  • accept：檔案上傳的檔案類型限制
  • name：欄位名稱（用於資料傳送）
  • id：元素識別碼（用於 label 關聯）
