1. HTML 簡介與基本結構

基礎

了解什麼是 HTML，以及 HTML 文件的基本結構。

程式碼範例：

<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的第一個網頁</title>
</head>
<body>
    <h1>歡迎來到我的網頁！</h1>
    <p>這是我的第一個 HTML 文件。</p>
</body>
</html>

詳細說明：

HTML (HyperText Markup Language) 是網頁的標準標記語言，用於創建網頁的結構和內容。

  • <!DOCTYPE html>：聲明這是 HTML5 文件
  • <html>：根元素，包含整個網頁
  • <head>：包含網頁的元資訊
  • <body>：包含網頁的可見內容
  • <title>：設定網頁標題（顯示在瀏覽器標籤）

每個 HTML 標籤都有開始標籤和結束標籤，內容放在兩者之間。

2. 文字標籤與格式化

基礎

學習各種文字標籤，包括標題、段落、強調等格式化元素。

程式碼範例：

<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <title>文字標籤範例</title>
</head>
<body>
    <!-- 標題標籤 -->
    <h1>這是最大的標題 (h1)</h1>
    <h2>這是第二級標題 (h2)</h2>
    <h3>這是第三級標題 (h3)</h3>
    <h4>這是第四級標題 (h4)</h4>
    <h5>這是第五級標題 (h5)</h5>
    <h6>這是最小的標題 (h6)</h6>

    <!-- 段落標籤 -->
    <p>這是一個段落。段落標籤會自動在前後加上空白行。</p>
    <p>這是另一個段落。可以包含很長的文字內容。</p>

    <!-- 文字格式化 -->
    <p>這段文字包含 <strong>重要內容</strong> 和 <em>強調文字</em>。</p>
    <p>這裡有 <b>粗體文字</b> 和 <i>斜體文字</i>。</p>
    <p>這是 <u>底線文字</u> 和 <s>刪除線文字</s>。</p>

    <!-- 特殊文字 -->
    <p>這是 <mark>螢光標記</mark> 文字。</p>
    <p>這是 <small>小字</small> 和 <sub>下標</sub> 以及 <sup>上標</sup>。</p>

    <!-- 引用 -->
    <blockquote>
        這是一個區塊引用。通常用於引用他人的話語或重要內容。
    </blockquote>

    <p>愛因斯坦說過：<q>想像力比知識更重要。</q></p>

    <!-- 程式碼 -->
    <p>使用 <code>console.log()</code> 來輸出訊息。</p>

    <pre>
        <code>
function sayHello() {
    console.log("Hello, World!");
}
        </code>
    </pre>

    <!-- 換行 -->
    <p>這是第一行<br>這是第二行</p>

    <!-- 水平線 -->
    <hr>

    <p>水平線上方的內容</p>
</body>
</html>

詳細說明：

HTML 提供了豐富的文字標籤來組織和格式化內容：

  • 標題標籤：h1 到 h6，用於建立內容層次
  • 段落標籤：p 用於組織文字段落
  • 語義化標籤：strong（重要）、em（強調）
  • 視覺化標籤：b（粗體）、i（斜體）、u（底線）
  • 特殊標籤：mark（標記）、code（程式碼）、pre（預格式化）

建議優先使用語義化標籤，這對 SEO 和無障礙設計更有幫助。

3. 連結與圖片

基礎

學習如何創建超連結和嵌入圖片，讓網頁更豐富有趣。

程式碼範例：

<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <title>連結與圖片範例</title>
</head>
<body>
    <h1>連結範例</h1>

    <!-- 外部連結 -->
    <p>訪問 <a href="https://www.google.com">Google</a> 搜尋引擎。</p>

    <!-- 在新視窗開啟連結 -->
    <p>在新視窗開啟 <a href="https://www.youtube.com" target="_blank">YouTube</a>。</p>

    <!-- 內部連結 -->
    <p><a href="#section1">跳到第一節</a></p>
    <p><a href="#section2">跳到第二節</a></p>

    <!-- 電子郵件連結 -->
    <p>聯絡我們：<a href="mailto:contact@example.com">contact@example.com</a></p>

    <!-- 電話連結 -->
    <p>撥打電話：<a href="tel:+886-2-1234-5678">02-1234-5678</a></p>

    <h1>圖片範例</h1>

    <!-- 基本圖片 -->
    <img src="image.jpg" alt="圖片描述">

    <!-- 設定圖片大小 -->
    <img src="logo.png" alt="公司標誌" width="200" height="100">

    <!-- 響應式圖片 -->
    <img src="photo.jpg" alt="風景照片" style="max-width: 100%; height: auto;">

    <!-- 圖片連結 -->
    <a href="https://www.example.com">
        <img src="banner.jpg" alt="點擊查看更多">
    </a>

    <!-- 圖片地圖 -->
    <img src="map.jpg" alt="地圖" usemap="#imagemap">
    <map name="imagemap">
        <area shape="rect" coords="0,0,100,100" href="page1.html" alt="區域1">
        <area shape="circle" coords="200,200,50" href="page2.html" alt="區域2">
    </map>

    <!-- 內容區塊 -->
    <h2 id="section1">第一節</h2>
    <p>這是第一節的內容...</p>

    <h2 id="section2">第二節</h2>
    <p>這是第二節的內容...</p>

    <!-- 回到頂部 -->
    <p><a href="#top">回到頂部</a></p>
</body>
</html>

詳細說明：

連結和圖片是網頁的重要元素：

  • <a> 標籤：創建超連結，href 屬性指定目標
  • target="_blank"：在新視窗開啟連結
  • <img> 標籤：嵌入圖片，src 指定圖片路徑
  • alt 屬性：圖片的替代文字，重要的無障礙功能
  • 錨點連結：使用 # 和 id 在頁面內跳轉

記得總是為圖片提供有意義的 alt 文字，這對 SEO 和無障礙設計很重要。

詳細說明：

清單是組織內容的重要工具：

  • <ul>：無序清單，使用項目符號
  • <ol>：有序清單，使用數字編號
  • <li>：清單項目
  • <dl>：描述清單，用於術語定義
  • <nav>：導航區塊，語義化標籤

清單可以巢狀使用，常用於創建導航選單和內容組織。

10. 進階函數概念

中級

學習更進階的函數概念，包括作用域、lambda 函數等。

程式碼範例：

# 變數作用域
global_var = "我是全域變數"

    local_var = "我是區域變數"
    print(f"函數內部：{global_var}")
    print(f"函數內部：{local_var}")

scope_example()
print(f"函數外部：{global_var}")
# print(local_var)  # 這會出錯，因為 local_var 不存在

def flexible_function(*args, **kwargs):
    print(f"位置參數：{args}")
    print(f"關鍵字參數：{kwargs}")

flexible_function(1, 2, 3, name="小明", age=25)

# 實用範例：計算平均值
def calculate_average(*numbers):
    if len(numbers) == 0:
        return 0
    return sum(numbers) / len(numbers)

avg1 = calculate_average(80, 90, 85)
avg2 = calculate_average(75, 88, 92, 79, 85)
print(f"平均分數1：{avg1}")
print(f"平均分數2：{avg2}")

# Lambda 函數（匿名函數）
square = lambda x: x ** 2
print(f"5的平方：{square(5)}")

# Lambda 與內建函數結合
numbers = [1, 2, 3, 4, 5]
squared_numbers = list(map(lambda x: x ** 2, numbers))
print(f"平方後：{squared_numbers}")

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(f"偶數：{even_numbers}")

# 高階函數範例
def apply_operation(numbers, operation):
    return [operation(x) for x in numbers]

numbers = [1, 2, 3, 4, 5]
doubled = apply_operation(numbers, lambda x: x * 2)
cubed = apply_operation(numbers, lambda x: x ** 3)

print(f"原數字：{numbers}")
print(f"乘以2：{doubled}")
print(f"立方：{cubed}")

# 遞迴函數範例
def factorial(n):
    """計算階乘"""
    if n <= 1:
        return 1
    return n * factorial(n - 1)

print(f"5! = {factorial(5)}")  # 120

詳細說明：

進階函數概念讓程式更靈活：

  • 作用域：變數的可見範圍
  • *args：接收任意數量的位置參數

        self.account_number = account_number
        self.owner = owner
        self.__balance = initial_balance  # 私有變數

    def deposit(self, amount):
        """存款"""
        if amount > 0:
            self.__balance += amount
            print(f"存款 ${amount}，餘額：${self.__balance}")
        else:
            print("存款金額必須大於0")

    def withdraw(self, amount):
        """提款"""
        if amount > 0 and amount <= self.__balance:
            self.__balance -= amount
            print(f"提款 ${amount}，餘額：${self.__balance}")
        else:
            print("提款失敗：金額無效或餘額不足")

    def get_balance(self):
        """查詢餘額"""
        return self.__balance

    def __str__(self):
        """字串表示"""
        return f"帳戶 {self.account_number}，持有人：{self.owner}"

    console.log(`- ${fruit}`);
}

while (count > 0) {
    console.log(`${count}...`);
    count--;
}
console.log("時間到！");

// do...while 迴圈
console.log("\n至少執行一次：");
let num = 0;
do {
    console.log(`數字：${num}`);
    num++;
} while (num < 3);

let numbers = [1, 3, 5, 8, 9, 10];

    if (number % 2 === 0) {
        console.log(`找到偶數：${number}`);
        break; // 跳出迴圈
    }
    console.log(`${number} 是奇數`);
}

console.log("\n只印出奇數：");
for (let i = 1; i <= 10; i++) {
    if (i % 2 === 0) {
        continue; // 跳過本次迭代
    }
    console.log(`奇數：${i}`);
}

// 巢狀迴圈
console.log("\n九九乘法表：");
for (let i = 1; i <= 3; i++) {
    let row = "";
    for (let j = 1; j <= 3; j++) {
        row += `${i}×${j}=${i*j} `;
    }
    console.log(row);
}

// 實用範例：計算總和
let scores = [85, 92, 78, 90, 88];
let total = 0;

    total += score;
}

let average = total / scores.length;
console.log(`\n成績總和：${total}`);
console.log(`平均分數：${average.toFixed(2)}`);

詳細說明：

JavaScript 迴圈的各種類型：

  • for：已知迭代次數的迴圈
  • for...of：遍歷可迭代物件（陣列、字串）
  • for...in：遍歷物件屬性
  • while：條件為真時持續執行
  • do...while：至少執行一次的迴圈

break 跳出迴圈，continue 跳過當前迭代。

async function robustFetch(url, options = {}) {
    const maxRetries = 3;
    let retries = 0;

    while (retries < maxRetries) {
        try {
            let response = await fetch(url, options);

            if (!response.ok) {
                throw new Error(`HTTP ${response.status}: ${response.statusText}`);
            }

            return await response.json();
        } catch (error) {
            retries++;
            console.log(`嘗試 ${retries}/${maxRetries} 失敗：`, error.message);

            if (retries >= maxRetries) {
                throw new Error(`在 ${maxRetries} 次嘗試後仍然失敗：${error.message}`);
            }

            // 等待後重試
            await new Promise(resolve => setTimeout(resolve, 1000 * retries));
        }
    }
}

5. 迴圈 (for/while/do-while)

程式碼範例：

public class Loops {
    public static void main(String[] args) {
        // for 迴圈 - 基本用法
        System.out.println("=== for 迴圈 ===");
        System.out.println("數字 1 到 5：");
        for (int i = 1; i <= 5; i++) {
            System.out.println("數字：" + i);
        }

        // for 迴圈 - 遍歷陣列
        String[] fruits = {"蘋果", "香蕉", "橘子", "葡萄"};
        System.out.println("\n水果清單：");
        for (int i = 0; i < fruits.length; i++) {
            System.out.println((i + 1) + ". " + fruits[i]);
        }

        // 增強型 for 迴圈 (for-each)
        System.out.println("\n使用 for-each：");
        for (String fruit : fruits) {
            System.out.println("- " + fruit);
        }

        // while 迴圈
        System.out.println("\n=== while 迴圈 ===");
        System.out.println("倒數計時：");
        int count = 5;
        while (count > 0) {
            System.out.println(count + "...");
            count--;
        }
        System.out.println("時間到！");

        // do-while 迴圈
        System.out.println("\n=== do-while 迴圈 ===");
        System.out.println("至少執行一次：");
        int num = 0;
        do {
            System.out.println("數字：" + num);
            num++;
        } while (num < 3);

        // 迴圈控制 - break 和 continue
        System.out.println("\n=== 迴圈控制 ===");
        System.out.println("尋找第一個偶數：");
        int[] numbers = {1, 3, 5, 8, 9, 10};

        for (int number : numbers) {
            if (number % 2 == 0) {
                System.out.println("找到偶數：" + number);
                break; // 跳出迴圈
            }
            System.out.println(number + " 是奇數");
        }

        System.out.println("\n只印出奇數：");
        for (int i = 1; i <= 10; i++) {
            if (i % 2 == 0) {
                continue; // 跳過本次迭代
            }
            System.out.println("奇數：" + i);
        }

        // 巢狀迴圈
        System.out.println("\n=== 巢狀迴圈 ===");
        System.out.println("九九乘法表（部分）：");
        for (int i = 1; i <= 3; i++) {
            for (int j = 1; j <= 3; j++) {
                System.out.print(i + "×" + j + "=" + (i * j) + " ");
            }
            System.out.println(); // 換行
        }

        // 實用範例：計算總和和平均
        int[] scores = {85, 92, 78, 90, 88};
        int total = 0;

        System.out.println("\n=== 計算總和和平均 ===");
        System.out.print("成績：");
        for (int score : scores) {
            System.out.print(score + " ");
            total += score;
        }

        double average = (double) total / scores.length;
        System.out.println("\n總分：" + total);
        System.out.printf("平均：%.2f%n", average);

        // 尋找最大值和最小值
        int max = scores[0];
        int min = scores[0];

        for (int score : scores) {
            if (score > max) {
                max = score;
            }
            if (score < min) {
                min = score;
            }
        }

        System.out.println("最高分：" + max);
        System.out.println("最低分：" + min);

        // 標籤迴圈（用於複雜的迴圈控制）
        System.out.println("\n=== 標籤迴圈 ===");
        outer: for (int i = 1; i <= 3; i++) {
            for (int j = 1; j <= 3; j++) {
                if (i == 2 && j == 2) {
                    System.out.println("跳出外層迴圈");
                    break outer; // 跳出標籤指定的迴圈
                }
                System.out.println("i=" + i + ", j=" + j);
            }
        }

        // 無限迴圈的安全使用
        System.out.println("\n=== 受控的無限迴圈 ===");
        int counter = 0;
        while (true) {
            counter++;
            System.out.println("計數：" + counter);

            if (counter >= 3) {
                System.out.println("達到條件，跳出迴圈");
                break;
            }
        }
    }
}

詳細說明：

Java 迴圈的各種類型和用法：

  • for：已知迭代次數的迴圈
  • for-each：遍歷陣列或集合的簡潔語法
  • while：條件為真時持續執行
  • do-while：至少執行一次的迴圈
  • break/continue：控制迴圈執行流程

選擇適當的迴圈類型能讓程式碼更清晰易懂。

            Console.WriteLine("\\n=== 邏輯運算 ===");
            Console.WriteLine($"{x} && {y} = {x && y}");  // AND
            Console.WriteLine($"{x} || {y} = {x || y}");  // OR
            Console.WriteLine($"!{x} = {!x}");            // NOT

            Console.ReadLine();
        }
    }
}

重點說明：

  • • 算術運算子：+、-、*、/、% (餘數)
  • • 比較運算子：==、!=、>、<、>=、<=
  • • 邏輯運算子：&& (AND)、|| (OR)、! (NOT)
  • • 賦值運算子：=、+=、-=、*=、/=
  • • 整數除法會截斷小數部分
